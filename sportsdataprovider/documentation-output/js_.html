app.js

I am the main application file. I watch a specified directory and when files have been updated or added I create a job so the file can be read then parsed and passed to the Transformer.js class which generates front end ready JSON files and publishes data to the Faye server.

<inner> onApplicationStart(message) → {void}

I start the application. I kick off the file watcher which in turn kicks off the job que (the Kue lib). Job que then starts the XML transformation process. In a nut shell: we read the incoming/updated XML and create JSON from that and write it to disk. After it's written to disk we then send a request to the Faye server with the JSON to update any clients which want to see the updates.
